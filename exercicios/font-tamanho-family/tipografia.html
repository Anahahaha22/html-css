Fontes css

Medidas Absolutas

  1. CM: Centimetro
  2. MM: Milimetro
  3. IN: Polegada
  4. PX: Pixels
  5. PT: Ponto
  6. PC: Paca

Medidas Relativas

  1. EM: Medida relativa ao tamanho atual da fonte.
  2. EX: Que é relativo ao tamanho X de uma fonte.
  3. REM: Relativo a fonte que está configurado no body.
  4. VW: View Width que quer dizer a largura da View Port (Ao tamanho da tela).
  5. VH: Altura da View Port (Ao tamanho da tela).
  6. %: Relativo á Porcentagem.

Recomendação oficial: Usar PX ou EM.

É bom usar o EM porque ele controla o tamanho da fonte para celular, tv, tablet, notebook etc...

Normalmente 16px = 1em

Normalmente os navegadores usam "Font-size: 16px;" tamanho padrão.
