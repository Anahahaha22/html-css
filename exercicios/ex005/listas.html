Listas em HTML5

Listas Ordenadas (Ordered List)

Exemplos de Ordered list são como receitas de bolos, listas de aprovados dos vestibular e lista de carros mais caros vendidos do mundo.

Para criar uma lista no HTML usamos a tag <ol> para demilitar a lista e <li> (list item) para identificar cada item da lista.


        
             <ol>
                 <li> Item 1 </li>
                 <li> Item 2 </li>
                 <li> Item 3 </li>
                 <li> Item 4 </li>
                 <li> Item 5 </li>
             </ol>

A Tag <ol> possui o parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esse parâmetro são:

  • 1- Valor padrão. Cria listas numeradas. Ex: 1, 2, 3, 4, ...
  • 2- Cria listas alfabéticas em maiúsculas. Ex: A, B, C, D, ...
  • 3- Cria listas alfabéticas em minuscúlas. Ex: a, b, c, d, ...
  • 4- Cria listas com algarismos romanos em maiúsculas. Ex: I, II, III, IV, ...
  • 5- Cria listas com algarismos romanos em minuscúlas. Ex: i, ii, iii, iv, ...


Você também pode indicar o ínicio da contagem usando o parametro start.

Por exemplo a tag <ol Type="I" start="5"> vai gerar itens numerados como V, VI, VII, VIII,IX ...

Listas não Ordenadas (Unordered Lists)

Para criar uma unordered list, vamos precisar usar a tag <ul> para deliminar a lista e tag <li> para criar um dos seus itens internos. Ex:

        
             <ul>
                 <li> Item 1 </li>
                 <li> Item 2 </li>
                 <li> Item 3 </li>
                 <li> Item 4 </li>
                 <li> Item 5 </li>
             </ul>

O marcador padrão é a bolinha preta totalmente preenchido (Circle), mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

  • Disc - Padrão. Uma bola preta totalmente pintada.
  • Circle - Uma bola com uma borda preta e sem preenchimento.
  • Square - Um pequeno quadrado preto totalmente pintado

    • Misturando as coisas

                
                 <ol>
                     <li> Item 1 </li>
                     <li> Item 2 </li>
                        <ol>
                            <li> Item 2.1 </li>
                            <li> Item 2.2 </li>
                            <li> Item 2.3 </li>
                        </ol>
                     <li> Item 3 </li>
                     <li> Item 4 </li>
                     <li> Item 5 </li>
                 </ol>

    Podemos também criar listas mistas, configurando listas dentro de outras listas. Veja o exemplo a seguir:

    1. Item 1
    2. Item 2
      1. Item 2.1
      2. Item 2.2
      3. Item 2.3
    3. Item 3
    4. Item 4
    5. Item 5

    Listas de Definições

    Todas as listas de Definições esta dentro da tag <dl< (Definition list). Cada termo é um <dt< (Definition Term) e cada descrição é um <dd< (Definition description).Ex:

            
            <dl>
                    <dt>HTML5<dt>
                    <dd>Linguagem de marcação ultilizado para criar conteúdo de sites.<dd>
                    <dt>CSS<dt>
                    <dd>Linguagem de marcação ultilizado para as especificação de estilos em sites.<dd>
                    <dt>JavaScript<dt>
                    <dd>Linguagem de programação para criar interatividades em sites.<dd>
            <dl>

    O resultado vai ser esse:


    HTML5
    Linguagem de marcação ultilizado para criar conteúdo de sites.
    CSS
    Linguagem de marcação ultilizado para as especificação de estilos em sites.
    JavaScript
    Linguagem de programação para criar interatividades em sites.
